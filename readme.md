<h1>Working:</h1>
<p>This is simple application that allows you to search for movies,seasons,episodes etc,<br>
and shows their details like: director, imdb rating, stars etc.</p>
<h1>FrontEnd</h1>
-> Angular <br>
-> Html <br>
-> Css <br>
-> Ionic Framework <br>
<h1>BackEnd</h1>
-> Nodejs <br>
-> Deployed on heroku<br>
<h1>Database</h1>
-> MongoDB <br>
-> Deployed on mlab.com
